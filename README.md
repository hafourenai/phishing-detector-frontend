# Phishing Detector Web UI

This repository provides a simple web-based user interface for the phishing detection system.

## Overview
The UI is designed to work with the backend from the following repository:  
https://github.com/hafourenai/phishing-detector-backend.git

It allows users to interact with the phishing detection service through a browser, without the need to run commands via the terminal.

## Technology
The interface is built using basic and widely supported web technologies:
- HTML
- CSS
- Vanilla JavaScript

## Purpose
The main goal of this project is to make the phishing detection backend more accessible, user-friendly, and easy to use for non-technical users by providing a lightweight web interface.
